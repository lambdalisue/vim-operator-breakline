# vim-operator-breakline

Like `gq` but break a line even on words.

## Example

```vim
map Gq <Plug>(operator-breakline-textwidth)
map GQ <Plug>(operator-breakline-manual)
```

## Requirements

- [kana/vim-operator-user](https://github.com/kana/vim-operator-user)
